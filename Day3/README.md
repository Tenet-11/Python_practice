## 1️⃣ 條件判斷（if / elif / else）

- 條件控制程式流程

### 學習重點

- 多條件判斷設計
- 巢狀條件判斷
- 邏輯清晰度

---

## 2️⃣ for 迴圈

### 核心理解

Python 的 for 不是 index-based 迴圈，
而是「遍歷 iterable」。

重點：

- 直接遍歷元素
- 使用 range 控制次數
- 使用 enumerate 同時取得索引與元素
- 使用 zip 同時遍歷多個序列

### 學習目標

- 熟悉基本迴圈結構
- 建立迭代思維
- 開始思考迴圈執行次數

---

## 3️⃣ while 迴圈

### 核心理解

- 依條件重複執行
- 適合未知次數的情境
- 必須避免無限迴圈

### 學習目標

- 理解條件控制與狀態變數
- 能設計停止條件

---

## 4️⃣ break 與 continue

- break：立即結束整個迴圈
- continue：跳過本次迴圈，進入下一次

### 學習重點

- 正確使用中斷邏輯
- 避免流程混亂

---

# 🧠 本日核心思維

- 迴圈是為了「處理重複邏輯」
- 條件判斷決定流程走向
- 控制流程是演算法的基礎

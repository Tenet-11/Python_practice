# 📘 學習內容

---

## 1️⃣ class 是什麼？

class 是「設計圖」。

它定義：

- 物件應該有哪些屬性
- 物件應該有哪些行為

就像：

- 汽車設計圖

- 人的結構設計圖

- 帳戶的設計圖

---

## 2️⃣ instance 是什麼？

instance 是根據 class 建立出來的「實際物件」。
(根據設計圖做出來的實體)

每個 instance：

- 有自己的資料
- 共用 class 定義的方法

---

## 3️⃣ __init__ 是什麼？

- 初始化方法
- 當物件被建立時自動執行
- 用來設定初始屬性

---

## 4️⃣ self 是什麼？

- 代表目前這個物件本身
- 用來存取該物件的屬性
- **是方法的第一個參數** (一定要加)

核心理解：

> 方法本質上只是函式，但第一個參數是物件本身。

---

## 5️⃣ Instance Variable

- 屬於某個物件
- 每個物件可以不同

---

## 6️⃣ 封裝（Encapsulation）

概念：

- 把資料與操作該資料的方法放在同一個 class
- 避免外部隨意修改內部狀態

---

# 🧠 本日核心思維

- class 是設計
- instance 是實體
- 方法操作屬性
- self 讓方法知道「我是誰」
